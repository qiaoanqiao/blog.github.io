变量

字符串

  1. 在定义字符串 赋值给变量, 这个变量gc.u.v.type=6为字符常亮 由此可得字符常亮类型. 不手动创建内存去指向.

赋值

变量赋值给另外一个变量 对于简单类型, 如: int null 会直接复制. 对于复杂类型 如 string object 会在写时复制

数组

  1. 数组变量
    • 数组有两个类型: packed Array(索引数组) 和 Hash Array(非索引数组) , packed 类型只有两个索引数组 而 packed 则 有八个索引数组.

- 当数组中的申键超过8个则会进行`处理`申请增加为`8+8`个桶数组
- `unset`并不会删除数组所占用的内存, 而是类型变为`is_undefined`
